# Generative-AI_Project
Googleエコシステム（Gemini, Google Workspace, GAS）を活用した業務効率化の実証ログです。

## 概要
本プロジェクトでは、インフラエンジニアとしての経験を活かし、生成AIを用いた業務プロセスの変革（AX）を推進するためのアイディアと実装サンプルを蓄積します。

## 100の活用アイディア（抜粋）
| No | アイディア名称 | 活用ツール | 期待される効果 |
|:---|:---|:---|:---|
| 1 | 未返信メールの自動リマインド | Gmail + GAS | 対応漏れの防止 |
| 11 | マニュアル作成支援 | Gemini + Docs | 属人化の解消 |
| 59 | 構成図からの設定値抽出 | Gemini | 設計作業の高速化 |
